# 开发环境搭建

## 1. 下载 Eigen
Eigen 是一个开源的 C++ 线性代数库，主要用于矩阵运算，包括矩阵、向量、线性方程组、特征值计算等。

## 2. 下载 Ceres-Solver
Ceres Solver 是一个开源的 C++ 非线性最小二乘优化库，主要用于解决非线性最小二乘问题。       

## 3. 下载 G2O      
G2O 是一个开源的 C++ 图优化库，主要用于解决图优化问题。
